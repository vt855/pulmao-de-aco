Pulmão de Aço — README multilíngue

Português (PT-BR)
-----------------
Pulmão de Aço — apoio e motivação para vencer vícios (fumo, álcool, drogas).
Arquivos:
- index.html
- styles.css
- script.js

Instruções rápidas:
1. Faça upload destes arquivos para o seu repositório GitHub (branch principal ou gh-pages).
2. O seletor "Language" usa o widget do Google Translate. Se a lista de idiomas não aparecer imediatamente, recarregue a página hospedada.
3. A introdução ("Venha vencer seus vícios aqui") é parte do mesmo HTML e será traduzida automaticamente pelo widget.
4. Para trocar imagens ou vídeos, edite as URLs em index.html.

English (EN)
------------
Pulmão de Aço — support and motivation to overcome addictions (smoking, alcohol, drugs).
Files:
- index.html
- styles.css
- script.js

Quick instructions:
1. Upload these files to your GitHub repository (main or gh-pages branch).
2. The "Language" selector uses the Google Translate widget. If the language list doesn't appear immediately, refresh the hosted page.
3. The intro ("Venha vencer seus vícios aqui") is part of the same HTML and will be automatically translated by the widget.
4. To change images or videos, edit the URLs in index.html.

Français (FR)
-------------
Pulmão de Aço — soutien et motivation pour vaincre les addictions (tabac, alcool, drogues).
Fichiers:
- index.html
- styles.css
- script.js

Instructions rapides:
1. Téléversez ces fichiers dans votre dépôt GitHub (branche principale ou gh-pages).
2. Le sélecteur "Language" utilise le widget Google Translate. Si la liste des langues n'apparaît pas immédiatement, actualisez la page hébergée.
3. L'intro ("Venha vencer seus vícios aqui") fait partie du même HTML et sera automatiquement traduite par le widget.
4. Pour changer les images ou vidéos, éditez les URLs dans index.html.

Notas:
- O widget do Google Translate depende de serviços externos e, raramente, pode demorar a inicializar.
- Se quiser que eu personalize textos em cada idioma manualmente (em vez de tradução automática), eu posso gerar versões estáticas traduzidas.
